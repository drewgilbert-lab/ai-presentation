# Lessons Learned: What Made This Work

<div class="grid grid-cols-2 gap-6 mt-6">
<div class="flex items-start gap-3">
<div class="flex-shrink-0 w-7 h-7 rounded-full bg-hg-royal text-white flex items-center justify-center font-bold text-[12px]">1</div>
<div>
<div class="font-bold text-hg-navy text-[13px]">Dashboard improved beyond the baseline</div>
<div class="text-[12px] text-hg-dark mt-1">Prompts tagged by use case, split into two campaigns. Set tightened to under 200, which improved data quality rather than just preserving volume.</div>
</div>
</div>
<div class="flex items-start gap-3">
<div class="flex-shrink-0 w-7 h-7 rounded-full bg-hg-royal text-white flex items-center justify-center font-bold text-[12px]">2</div>
<div>
<div class="font-bold text-hg-navy text-[13px]">Title recommendation engine built for Refresh Grid</div>
<div class="text-[12px] text-hg-dark mt-1">Recommends titles by comparing existing content against gaps and competitor citation topics. Used to plan 15 Q2 titles.</div>
</div>
</div>
<div class="flex items-start gap-3">
<div class="flex-shrink-0 w-7 h-7 rounded-full bg-hg-royal text-white flex items-center justify-center font-bold text-[12px]">3</div>
<div>
<div class="font-bold text-hg-navy text-[13px]">Custom internal linking process</div>
<div class="text-[12px] text-hg-dark mt-1">Reads only relevant blog content. Reviews sitemap across use case, product, and blog pages, finds opportunities, and inserts recommendations for marketing review.</div>
</div>
</div>
<div class="flex items-start gap-3">
<div class="flex-shrink-0 w-7 h-7 rounded-full bg-hg-royal text-white flex items-center justify-center font-bold text-[12px]">4</div>
<div>
<div class="font-bold text-hg-navy text-[13px]">Python + Claude Sonnet extends the platform</div>
<div class="text-[12px] text-hg-dark mt-1">Pairing deterministic Python with Claude Sonnet inside AirOps lets us build past off-the-shelf functionality. Pulling data through AirOps' own API is now our default approach.</div>
</div>
</div>
<div class="flex items-start gap-3 col-span-2">
<div class="flex-shrink-0 w-7 h-7 rounded-full bg-hg-red text-white flex items-center justify-center font-bold text-[12px]">!</div>
<div>
<div class="font-bold text-hg-navy text-[13px]">Elementor requires careful DOM-level scripting for WordPress push</div>
<div class="text-[12px] text-hg-dark mt-1">HG's site stores serialized page-builder data, not clean post HTML. A naive write can break layouts. Mitigation: control the integration with proper HTML/DOM scripting in Python, parse and write into Elementor's structure deliberately, validate on staging first. We are planning for this, not discovering it mid-build.</div>
</div>
</div>
</div>
